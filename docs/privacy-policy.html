CurveCue Privacy Policy

Last updated: May 25, 2026

CurveCue is a driving-assistance app that provides curve narration, route analysis, and related map features. This policy describes what data the app accesses, how it is used, and what choices users have.

Who This Policy Applies To

This policy applies to the Android app CurveCue published by Mustafa Hathiyari on the Google Play Store.

Data the App Accesses

  • Precise location (GPS) while a driving session is active
  • Approximate location (network-based) as a fallback
  • Search queries entered by the user for destinations
  • Route and destination coordinates needed to calculate or preview a route
  • GPX track files selected by the user for route import
  • App settings and recent destinations stored locally on the device
  • Session event logs (timestamps, speed samples, curve events) written to device storage and optionally shared by the user via Android's share sheet
  • Basic diagnostic information generated by the Android platform and libraries

How the App Uses Data

  • To determine the user's current position during an active driving session
  • To time curve narration and route progress updates
  • To search for destinations and display human-readable place names
  • To calculate routes, including online fallback routing when offline data is unavailable
  • To import user-selected GPX track files as route sources
  • To log session events (speed, curve cues, timing) to a local file for optional post-ride review
  • To download and cache map tiles and region data selected by the user
  • To save app preferences and recent destinations on the device

Data Sharing and Network Services

CurveCue does not sell personal data.

When certain features are used, limited data may be sent to third-party map or routing services in order to fulfill the request:

  • Destination search and reverse geocoding requests may be sent to the Nominatim service operated by OpenStreetMap contributors.
  • Online route requests may be sent to the OSRM routing service.
  • Map tile requests may be sent to OpenStreetMap tile servers or equivalent endpoints when online map tiles are used.

Those services receive only the request data needed to respond—such as search text, coordinates, and standard HTTP request metadata. Refer to those services' own privacy policies for how they handle request data.

Session event logs and GPX files are never transmitted by the app. They remain on-device unless the user explicitly shares them using Android's share sheet.

Data Storage

CurveCue primarily stores data locally on the user's device. Local data may include:

  • Driving mode and audio/settings preferences
  • Disclaimer acknowledgement
  • Recent destinations
  • Downloaded region packages and cached map tiles
  • Session event log files (CSV) stored in app-private storage; these are accessible to the user via the Android share sheet and are not transmitted anywhere by the app

CurveCue does not provide user accounts, cloud backup, or server-side profile storage.

Data Retention

Most app data remains on the device until the user clears app data, uninstalls the app, or removes downloaded data from within the app where supported. Network providers used for search, routing, or tile delivery may retain request logs according to their own policies.

Children

CurveCue is not directed to children under 13. We do not knowingly collect personal information from children.

Security

All network requests made by the app use HTTPS. Reasonable efforts are made to reduce unnecessary data collection and to store user preferences locally when possible. No method of transmission or storage is completely secure, and absolute security cannot be guaranteed.

User Choices

  • Users can deny location permission, though core driving-session features will not function correctly without it.
  • Users can decline to use destination search or online routing features, using offline-only mode instead.
  • Users can clear app data or uninstall the app to remove all locally stored preferences, recents, and session logs.
  • Session log files can be deleted individually from within the app's settings or by clearing app data.

Changes to This Policy

This policy may be updated as the app changes. The most current version is published at the privacy policy URL listed on the Google Play store listing.
